Crying Atoms

This is a chemistry education game under a Sci-Fi background based on Bitsy. Designed by me, Jiahong Li. The game is in development. It's currently at the tutorial stage. Current game length is about 5 mins. My design goal is a game with 1-hour game play.

Game Control: Use W, A, S, D or ↑, ←, ↓, → to control the avatar. Walk into NPCs to interact.
